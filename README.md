# Household Service App

A web application that connects customers with professional service providers for various household needs. Built using **Flask**, the app supports three distinct user roles: **Admin**, **Customer**, and **Professional**.

## 📌 Project Overview

This project is developed as part of an academic requirement at IIT Madras. It aims to offer an intuitive platform for booking, managing, and fulfilling household services like cleaning, plumbing, and more.

## 🛠 Frameworks and Libraries

- **Backend:** Flask  
- **Database:** Flask-SQLAlchemy  
- **Authentication:** Flask-Login  
- **File Handling:** Werkzeug  
- **Frontend:** HTML, CSS, Bootstrap (structure implied)

## 🔑 User Roles & Features

### 🛠 Admin
- Approve or reject professionals
- Block/unblock users
- Add/edit/delete services
- Manage users and requests

### 👥 Customer
- Browse and book services
- Manage requests
- Rate and review services
- Edit or delete pending requests

### 🧑‍🔧 Professional
- Create profile
- Accept/reject/complete requests
- View customer reviews
